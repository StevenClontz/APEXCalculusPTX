Theorem 13.8.13. Second Derivative Test.

Let \(R\) be an open set on which a function \(z=f(x,y)\) and all its first and second partial derivatives are defined, let \(P = (x_0,y_0)\) be a critical point of \(f\) in \(R\text{,}\) and let

\begin{equation*} D = f_{xx}(x_0,y_0)f_{yy}(x_0,y_0)-f_{xy}^{\,2}(x_0,y_0)\text{.} \end{equation*}

  1. If \(D \gt 0\) and \(f_{xx}(x_0,y_0) \gt 0\text{,}\) then \(f\) has a relative minimum at \(P\text{.}\)

  2. If \(D \gt 0\) and \(f_{xx}(x_0,y_0)\lt 0\text{,}\) then \(f\) has a relative maximum at \(P\text{.}\)

  3. If \(D\lt 0\text{,}\) then \(f\) has a saddle point at \(P\text{.}\)

  4. If \(D=0\text{,}\) the test is inconclusive.
