Let \(f\) be a function defined on \((-\infty,b)\) for some number \(b\text{.}\) The limit of \(f\) at negative infinity is \(L\text{,}\) denoted \(\lim_{x\to-\infty} f(x)=L\text{,}\) if for every \(\epsilon \gt 0\) there exists \(M\lt b\) such that if \(x \lt M\text{,}\) then \(\abs{f(x)-L}\lt \epsilon\text{.}\)
