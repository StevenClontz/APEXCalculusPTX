Theorem 3.3.5. Test For Increasing/Decreasing Functions.

Let \(f\) be a continuous function on \([a,b]\) and differentiable on \((a,b)\text{.}\)

  1. If \(\fp(c) \gt 0\) for all \(c\) in \((a,b)\text{,}\) then \(f\) is increasing on \([a,b]\text{.}\)

  2. If \(\fp(c) \lt 0\) for all \(c\) in \((a,b)\text{,}\) then \(f\) is decreasing on \([a,b]\text{.}\)

  3. If \(\fp(c) =0\) for all \(c\) in \((a,b)\text{,}\) then \(f\) is constant on \([a,b]\text{.}\)

The conclusions of Item 1 and Item 2 also hold if \(\fp(c) = 0\) for a finite number of nonadjacent values of \(c\) in \(I\text{.}\)
