Solution 6.1.23.1.

This integral requires two different methods to evaluate it. We get to those methods by splitting up the integral into two terms:

\begin{equation*} \int \frac{4-x}{\sqrt{16-x^2}}\, dx = \int \frac{4}{\sqrt{16-x^2}}\, dx - \int \frac{x}{\sqrt{16-x^2}}\, dx\text{.} \end{equation*}

We handle each separately. The first integral is handled using a straightforward application of TheoremÂ 6.1.20:

\(\ds \int \frac{4}{\sqrt{16-x^2}}\, dx = 4\sin^{-1}\left(\frac{x}{4}\right) + C\text{.}\)

The second integral is handled by substitution, with \(u = 16-x^2\text{.}\) \(\ds \int\frac{x}{\sqrt{16-x^2}}\, dx\text{:}\) Set \(u = 16-x^2\text{,}\) so \(du = -2x\, dx\) and \(x\, dx = -du/2\text{.}\) We have

\begin{align*} \int\frac{x}{\sqrt{16-x^2}}\, dx \amp = \int\frac{-du/2}{\sqrt{u}}\\ \amp = -\frac12\int \frac{1}{\sqrt{u}}\, du\\ \amp = - \sqrt{u} + C\\ \amp = -\sqrt{16-x^2} + C\text{.} \end{align*}

Combining these together, we have

\begin{equation*} \int \frac{4-x}{\sqrt{16-x^2}}\, dx = 4\sin^{-1}\left(\frac{x}{4}\right) + \sqrt{16-x^2}+C\text{.} \end{equation*}

As with all definite integrals, you can check your work by differentiation.
