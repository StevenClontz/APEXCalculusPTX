Exercise Group 35–38.

In the following exercises, a solid is described along with its density function. Find the center of mass of the solid using spherical coordinates. (Note: these are the same solids and density functions as found in Exercises 14.7.3.31 through Exercise 14.7.3.34.)

35.

The upper half of the unit ball, bounded between \(z= 0\) and \(z=\sqrt{1-x^2-y^2}\text{,}\) with density function \(\delta(x,y,z) =1\text{.}\)

36.

The spherical shell bounded between \(x^2+y^2+z^2=16\) and \(x^2+y^2+z^2=25\) with density function \(\delta(x,y,z) = \sqrt{x^2+y^2+z^2}\text{.}\)

37.

The conical region bounded above \(z=\sqrt{x^2+y^2}\) and below the sphere \(x^2+y^2+z^2=1\) with density function \(\delta(x,y,z) = z\text{.}\)

38.

The cone bounded above \(z=\sqrt{x^2+y^2}\) and below the plane \(z=1\) with density function \(\delta(x,y,z) = z\text{.}\)
