Let \(\vec r(t)\) be a vector-valued function defined on an open interval \(I\) containing \(c\text{.}\)

  1. \(\vec r(t)\) is continuous at \(c\) if \(\lim\limits_{t\to c} \vec r(t) = r(c)\text{.}\)

  2. If \(\vec r(t)\) is continuous at all \(c\) in \(I\text{,}\) then \(\vec r(t)\) is continuous on \(I\text{.}\)
