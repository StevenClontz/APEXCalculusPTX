  1. The Power Rule states that if \(f(x) = x^3\text{,}\) then \(\fp(x) = 3x^2\text{.}\)

  2. To find the equation of the line tangent to the graph of \(f\) at \(x=-1\text{,}\) we need a point and the slope. The point is \((-1,f(-1)) = (-1, -1)\text{.}\) The slope is \(\fp(-1)= 3\text{.}\) Thus the tangent line has equation \(y = 3(x-(-1))+(-1) = 3x+2\text{.}\)

  3. We can use the tangent line to approximate \((-1.1)^3\) since \(-1.1\) is close to \(-1\text{.}\) We have

    \begin{equation*} (-1.1)^3 \approx 3(-1.1)+2 = -1.3\text{.} \end{equation*}

    We can easily find the actual value: \((-1.1)^3 = -1.331\text{.}\)
