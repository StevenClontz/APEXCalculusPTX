Solution 9.8.14.1.

The following argument is essentially the same as that given for \(f(x) = \cos(x)\) in ExampleÂ 9.8.9.

Given a value \(x\text{,}\) the magnitude of the error term \(R_n(x)\) is bounded by

\begin{equation*} \abs{R_n(x)} \leq \frac{\max\abs{\,f^{(n+1)}(z)}}{(n+1)!}\abs{x^{(n+1)}}\text{.} \end{equation*}

Since all derivatives of \(\sin(x)\) are \(\pm \cos(x)\) or \(\pm\sin(x)\text{,}\) whose magnitudes are bounded by \(1\text{,}\) we can state

\begin{equation*} \abs{R_n(x)} \leq \frac{1}{(n+1)!}\abs{x^{(n+1)}}\text{.} \end{equation*}

For any \(x\text{,}\) \(\lim\limits_{n\to\infty} \frac{x^{n+1}}{(n+1)!} = 0\text{.}\) Thus by the Squeeze Theorem, we conclude that \(\lim\limits_{n\to\infty} R_n(x) = 0\) for all \(x\text{,}\) and hence

\begin{equation*} \sin(x) = \infser[0] (-1)^{n}\frac{x^{2n+1}}{(2n+1)!} \text{ for all \(x\) }\text{.} \end{equation*}
