The following limits hold.

  1. Constants: \(\lim\limits_{(x,y)\to (x_0,y_0)} b = b\)

  2. Identity \(\lim\limits_{(x,y)\to (x_0,y_0)} x = x_0\text{;}\) \(\lim\limits_{(x,y)\to (x_0,y_0)} y = y_0\)

  3. Sums/Differences: \(\lim\limits_{(x,y)\to (x_0,y_0)}\big(f(x,y)\pm g(x,y)\big) = L\pm K\)

  4. Scalar Multiples: \(\lim\limits_{(x,y)\to (x_0,y_0)} b\cdot f(x,y) = bL\)

  5. Products: \(\lim\limits_{(x,y)\to (x_0,y_0)} f(x,y)\cdot g(x,y) = LK\)

  6. Quotients: \(\lim\limits_{(x,y)\to (x_0,y_0)} f(x,y)/g(x,y) = L/K\text{,}\) (\(K\neq 0)\)

  7. Powers: \(\lim\limits_{(x,y)\to (x_0,y_0)} f(x,y)^n = L^n\)
