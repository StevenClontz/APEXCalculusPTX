While it is not specified which order we are to use, we will evaluate the double integral using both orders to help drive home the point that it does not matter which order we use.

Using the order \(dy\, dx\text{:}\) The bounds on \(y\) go from “curve to curve,” i.e., \(0\leq y\leq 1-x/2\text{,}\) and the bounds on \(x\) go from “point to point,” i.e., \(0\leq x\leq 2\text{.}\)

\begin{align*} \iint_R (3xy-x^2-y^2+6\big)\, dA \amp = \int_0^2\int_0^{-\frac x2+1} (3xy-x^2-y^2+6\big)\, dy\, dx\\ \amp = \int_0^2\left.\left(\frac32xy^2-x^2y-\frac13y^3+6y\right)\right|_0^{-\frac x2+1}dx\\ \amp = \int_0^2 \left(\frac{11}{12}x^3-\frac{11}{4}x^2-x+\frac{17}3\right)dx\\ \amp = \left.\left(\frac{11}{48}x^4-\frac{11}{12}x^3-\frac12x^2+\frac{17}3x\right)\right|_0^2\\ \amp = \frac{17}3=5.\overline{6}\text{.} \end{align*}

Now lets consider the order \(dx \, dy\text{.}\) Here \(x\) goes from “curve to curve,” \(0\leq x\leq 2-2y\text{,}\) and \(y\) goes from “point to point,” \(0\leq y\leq 1\text{:}\)

\begin{align*} \iint_R (3xy-x^2-y^2+6\big)\, dA \amp = \int_0^1\int_0^{2-2y} (3xy-x^2-y^2+6\big)\, dx\, dy\\ \amp = \int_0^1\left.\left(\frac32x^2y-\frac13x^3-xy^2+6x\right)\right|_0^{2-2y} dy\\ \amp = \int_0^1\left(\frac{32}3y^3-22y^2+2y+\frac{28}3\right)dy\\ \amp =\left.\left(\frac83y^4-\frac{22}3y^3+y^2+\frac{28}3y\right)\right|_0^1\\ \amp =\frac{17}3=5.\overline{6}\text{.} \end{align*}

We obtained the same result using both orders of integration.
