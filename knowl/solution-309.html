Solution 6.1.13.1.

This example employs a wonderful trick: multiply the integrand by “1” so that we see how to integrate more clearly. In this case, we write “1” as

\begin{equation*} 1 = \frac{\sec(x) + \tan(x) }{\sec(x) + \tan(x) }\text{.} \end{equation*}

This may seem like it came out of left field, but it works beautifully. Consider:

\begin{align*} \int \sec(x) \, dx \amp = \int \sec(x) \cdot \frac{\sec(x) + \tan(x) }{\sec(x) + \tan(x) }\, dx\\ \amp = \int \frac{\sec^2(x) + \sec(x) \tan(x) }{\sec(x) + \tan(x) }\, dx.\\ \end{align*}

Now let \(u = \sec(x) +\tan(x)\text{;}\) this means \(du = (\sec(x) \tan(x) + \sec^2(x) )\, dx\text{,}\) which is our numerator. Thus:

\begin{align*} \amp = \int \frac{du}{u}\\ \amp = \ln\abs{u} + C\\ \amp = \ln\abs{\sec(x) +\tan(x) } + C\text{.} \end{align*}
