Let \(C\) be a smooth curve in the plane or in space.

  1. If \(C\) is defined by \(y=f(x)\text{,}\) then

    \begin{equation*} \kappa = \frac{\abs{\fp'(x)}}{\Big(1+\big(\fp(x)\big)^2\Big)^{3/2}}\text{.} \end{equation*}

  2. If \(C\) is defined as a vector-valued function in the plane, \(\vrt = \la x(t), y(t)\ra\text{,}\) then

    \begin{equation*} \kappa = \frac{\abs{x'y''-x''y'}}{\big((x')^2+(y')^2\big)^{3/2}}\text{.} \end{equation*}

  3. If \(C\) is defined in space by a vector-valued function \(\vrt\text{,}\) then

    \begin{equation*} \kappa = \frac{\norm{\unittangentprime(t)}}{\norm{\vrp(t)}} = \frac{\norm{\vrp(t)\times\vrpp(t)}}{\norm{\vrp(t)}^3} = \frac{\vec a(t)\cdot \unitnormal(t)}{\norm{\vec v(t)}^2}\text{.} \end{equation*}
