When composing functions, we need to make sure that the new function is actually defined. For instance, consider \(f(x) = \sqrt{x}\) and \(g(x) = -x^2-1\text{.}\) The domain of \(f\) excludes all negative numbers, but the range of \(g\) is only negative numbers. Therefore the composition \(f(g(x)) = \sqrt{-x^2-1}\) is not defined for any \(x\text{,}\) and hence is not differentiable.
