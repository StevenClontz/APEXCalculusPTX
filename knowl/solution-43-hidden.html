Let \(\varepsilon \gt 0\) be given. We wish to find \(\delta \gt 0\) such that when \(\abs{x-0}\lt\delta\text{,}\) \(\abs{f(x)-0}\lt\varepsilon\text{.}\)

First, some preliminary investigation to find a suitable \(\delta\text{.}\) Consider \(\abs{f(x)-0}\lt\varepsilon\text{:}\)

\begin{equation*} \begin{aligned} \abs{\left(e^{2x}-1\right)-0}\amp\lt\varepsilon\\ \abs{e^{2x}-1}\amp\lt\varepsilon\\ -\varepsilon\amp\lt e^{2x}-1\lt\varepsilon\\ 1-\varepsilon\amp\lt e^{2x}\lt1+\varepsilon\\ \ln(1-\varepsilon)\amp\lt 2x\lt\ln(1+\varepsilon)\\ \frac{1}{2}\ln(1-\varepsilon)\amp\lt x\lt\frac{1}{2}\ln(1+\varepsilon)\\ \frac{1}{2}\ln(1-\varepsilon)\amp\lt x-0\lt\frac{1}{2}\ln(1+\varepsilon) \end{aligned} \end{equation*}

Since we will need to start with \(\abs{x-0}\lt\delta\text{,}\) this suggests we take \(\delta=\min\left\{\frac{1}{2}\abs{\ln(1-\varepsilon)},\frac{1}{2}\ln(1+\varepsilon)\right\}=\frac{1}{2}\ln(1+\varepsilon)\text{.}\)

Now we can apply the definition.

\begin{equation*} \begin{aligned} \abs{x-0}\amp\lt\delta\\ \abs{x}\amp\lt\frac{1}{2}\ln(1+\varepsilon)\lt\frac{1}{2}\abs{\ln(1-\varepsilon)}\\ \frac{1}{2}\ln(1-\varepsilon)\amp\lt x\lt\frac{1}{2}\ln(1+\varepsilon)\\ \ln(1-\varepsilon)\amp\lt 2x\lt\ln(1+\varepsilon)\\ 1-\varepsilon\amp\lt e^{2x}\lt1+\varepsilon\\ -\varepsilon\amp\lt e^{2x}-1\lt\varepsilon\\ \abs{e^{2x}-1}\amp\lt\varepsilon\\ \abs{\left(e^{2x}-1\right)-0}\amp\lt\varepsilon \end{aligned} \end{equation*}

In other words, \(\abs{x-0}\lt\delta\) implies \(\abs{\left(e^{2x}-1\right)-0}\lt\varepsilon\text{.}\) This is what we needed to prove.
