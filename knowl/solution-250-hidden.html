1. \(y = \sin(x)\text{:}\) As \(f(x) = \sin(x)\text{,}\) \(\fp(x) = \cos(x)\text{.}\) Thus

    \begin{equation*} dy = \cos(x)dx\text{.} \end{equation*}

  2. \(y = e^x\left(x^2+2\right)\text{:}\) Let \(f(x) = e^x\left(x^2+2\right)\text{.}\) We need \(\fp(x)\text{,}\) requiring the Theorem 2.4.2.

    We have \(\fp(x) = e^x\left(x^2+2\right) + 2xe^x\text{,}\) so

    \begin{equation*} dy = \left(e^x\left(x^2+2\right) + 2xe^x\right)dx\text{.} \end{equation*}

  3. \(y = \sqrt{x^2+3x-1}\text{:}\) Let \(f(x) = \sqrt{x^2+3x-1}\text{;}\) we need \(\fp(x)\text{,}\) requiring the Theorem 2.5.4.

    We have \(\fp(x) = \frac{1}{2}\left(x^2+3x-1\right)^{-\frac{1}{2}}(2x+3) = \frac{2x+3}{2\sqrt{x^2+3x-1}}\text{.}\) Thus

    \begin{equation*} dy = \frac{(2x+3)dx}{2\sqrt{x^2+3x-1}}\text{.} \end{equation*}
