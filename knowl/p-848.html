Let \(f\) be a function defined on an open interval \(I\) containing \(c\text{.}\)

  1. \(f\) is continuous at a point \(c\) if \(\lim\limits_{x\to c}f(x) = f(c)\text{.}\)

  2. \(f\) is continuous on an open interval \(I\) if \(f\) is continuous at \(c\) for all values of \(c\) in \(I\text{.}\) If \(f\) is continuous on \((-\infty,\infty)\text{,}\) we say \(f\) is continuous everywhere (or everywhere continuous).
