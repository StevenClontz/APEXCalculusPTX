Solution 12.4.5.1.

We find \(\vrp(t) = \la 2t-1,2t+1\ra\text{,}\) and

\begin{equation*} \norm{\vrp(t)} = \sqrt{(2t-1)^2+(2t+1)^2} = \sqrt{8t^2+2}\text{.} \end{equation*}

Therefore

\begin{equation*} \unittangent(t) = \frac{1}{\sqrt{8t^2+2}}\la 2t-1,2t+1\ra = \la \frac{2t-1}{\sqrt{8t^2+2}},\frac{2t+1}{\sqrt{8t^2+2}}\ra\text{.} \end{equation*}

When \(t=0\text{,}\) we have \(\unittangent(0) = \la -1/\sqrt{2},1/\sqrt{2}\ra\text{;}\) when \(t=1\text{,}\) we have \(\unittangent(1) = \la 1/\sqrt{10}, 3/\sqrt{10}\ra\text{.}\) We leave it to the reader to verify each of these is a unit vector. They are plotted in Figure 12.4.6
