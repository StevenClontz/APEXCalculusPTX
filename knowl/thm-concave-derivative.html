Theorem 3.4.4.

Let \(f\) be a continuous function on \([a,b]\) and differentiable on \((a,b)\text{.}\)

  1. If \(\fpp(c) \gt 0\) for all \(c\) in \((a,b)\text{,}\) then \(f\) is concave up on \([a,b]\text{.}\)

  2. If \(\fpp(c) \lt 0\) for all \(c\) in \((a,b)\text{,}\) then \(f\) is concave down on \([a,b]\text{.}\)

  3. If \(\fpp(c) =0\) for all \(c\) in \((a,b)\text{,}\) then \(f\) is linear on \([a,b]\text{.}\)
