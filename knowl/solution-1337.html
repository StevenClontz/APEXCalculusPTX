Solution 13.8.3.17.1.

The region has no “corners” or “vertices,” just a smooth edge.

To find critical points along the circle \(x^2+y^2=4\text{,}\) we solve for \(y^2\text{:}\) \(y^2=4-x^2\text{.}\) We can go further and state \(y=\pm\sqrt{4-x^2}\text{.}\)

We can rewrite \(f\) as \(f(x)=x^2+2x + (4-x^2) + 2\sqrt{4-x^2} = 2x+4+2\sqrt{4-x^2}\text{.}\) (We will return and use \(-\sqrt{4-x^2}\) later.) Solving \(f\,'(x)=0\text{,}\) we get \(x=\sqrt{2} \Rightarrow y=\sqrt{2}\text{.}\) \(f\,'(x)\) is also undefined at \(x=\pm 2\text{,}\) where \(y=0\text{.}\)

Using \(y=-\sqrt{4-x^2}\text{,}\) we rewrite \(f(x,y)\) as \(f(x) = 2x+4-2\sqrt{4-x^2}\text{.}\) Solving \(\fp(x) =0\text{,}\) we get \(x=-\sqrt{2},\, y=-\sqrt{2}\text{.}\) Again, \(\fp(x)\) is undefined at \(x=\pm 2\text{.}\)

The function \(f(x,y)\) itself has a critical point at \((-1,-1)\text{.}\)

Checking the value of \(f\) at \((-1,-1)\text{,}\) \((\sqrt{2},\sqrt{2})\text{,}\) \((-\sqrt{2},-\sqrt{2})\text{,}\) \((2,0)\) and \((-2,0)\text{,}\) we find the absolute maximum is at \((\sqrt2,\sqrt2,4+4\sqrt2)\) and the absolute minimum is at \((-1,-1,-2)\text{.}\)
