Figure 1.1.13 shows values of \(f(x)\) for values of \(x\) near \(0\text{.}\) It is clear that as \(x\) takes on values very near \(0\text{,}\) \(f(x)\) takes on values very near \(1\text{.}\) It turns out that if we let \(x=0\) for either “piece” of \(f(x)\text{,}\) \(1\) is returned; this is significant and we'll return to this idea later.
